# Quick Translate

웹페이지에서 텍스트를 드래그해 선택하면 즉시 한국어로 번역해주는 Chrome 확장 프로그램입니다.  
더블클릭으로 단어, 트리플클릭으로 문장을 빠르게 선택할 수 있습니다.

## 주요 기능

- 텍스트 선택 시 팝업 또는 플로팅 윈도우로 번역 표시
- 번역 기록 자동 저장 (검색 / 정렬 / 필터)
- 단어별 학습 상태 관리, 즐겨찾기, 플래시카드 퀴즈
- 영어 동사 활용형 및 유의어 자동 표시
- 라이트 / 다크 / 시스템 테마
- `Option+T`로 확장 전체 활성/비활성 토글

## 설치

1. `chrome://extensions` → 개발자 모드 활성화
2. **압축해제된 확장 프로그램을 로드합니다** → 프로젝트 폴더 선택

## API 키 설정

[Google Cloud Console](https://console.cloud.google.com/apis/library/translate.googleapis.com)에서 Cloud Translation API를 활성화하고 발급받은 키를 설정 페이지에 입력하세요.
